coding standard file naming conventions

- ${number}0${number}

e.g 1001 means 10 as major value with 0 as separator and 1 as minor value

major values:

- 1 - backend
- 2 - frontend
- 3 - contract
- 4 - general project structure
- 5 - general coding standards
- 6 - documentations
- 7 - collaboration

When you need to add a new coding standard, open an issue first and submit a pull request containing the proposed coding standard.

when you see a conflicting convention within the coding standards, open an issue and submit a pull request with the proposed changes.
